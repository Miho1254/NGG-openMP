**D. OnGameModeInit hook — Spawn objects & NPC:**
- Tạo object 19473 (cây cần sa) tại 25 vị trí
- Tạo 3D label tại khu chế biến: `"{FF6347}Khu che bien can sa\n{FFFFFF}/chebiencansa"`
- Tạo NPC actor ID 116 tại -2204.5874, 961.0796, 80.0 + 3D label: `"{FF6347}Nguoi thu mua\n{FFFFFF}/bancansa"`

**E. `/haican` command:**
- Kiểm tra player có gần cây nào không (range 3.0)
- Kiểm tra cây còn sống
- Đóng băng player 5 giây + animation BOM_Plant
- Timer callback:
  - 70%: +1 Cần sa thô
  - 20%: +2 Cần sa thô + message vui
  - 10%: -20 HP + message "Bị rắn cắn!"
- Destroy object, set alive = false, start respawn timer

**F. Respawn timer callback:**
- Recreate object
- Set alive = true

**G. `/chebiencansa` command:**
- Kiểm tra gần khu chế biến (label position)
- Kiểm tra có Cần sa thô trong balo
- Đóng băng 5 giây + animation BD_FIRE
- Random 1-5 túi zip:
  - 1 túi: message khịa
  - 5 túi: message chúc mừng
- Xóa Cần sa thô, thêm Túi zip

**H. `/bancansa` command:**
- Kiểm tra gần NPC (range 5.0)
- Kiểm tra có Túi zip trong balo
- Dialog chọn số lượng (nhập số)
- Bán: xóa túi zip, +pDirtyMoney (không phải pCash)
- Message cho player: "Ban nhan duoc $X tien ban"

**I. LAW notification hook:**
- Khi giao dịch hoàn tất, gửi message cho toàn bộ LAW:
  `"[CAMERA] Vua phat hien giao dich chat cam cua PLAYER_NAME, cac don vi luu y doi tuong nay]"`
- KHÔNG set wanted

**J. `/tuido` integration:**
- Hiển thị pDirtyMoney trong `/tuido` hoặc `/stats`
- Dirty money dùng để mua đồ ở chợ đen (tương lai)
